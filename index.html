Ryerson Cansat: September 2017- September 2020 (Toronto, ON)

Team Captain

    Ryerson Cansat is an engineering design team at Ryerson that participates in the NASA sponsored International CanSat competition in Blacksburg, Virginia.

    The main objective of the CanSat competition is to design a miniature satellite- usually the size of a soda can. The main functions of the satellite include obtaining the altitude, air pressure, temperature, time, GPS longitude and latitude as well as the software state (which is essentially the operating state of the software- launch, ascent, rocket separation, descent, etc.) You can learn more about the specifics of the competition here: CANSAT.

    I have been on the electronics team of Ryerson CanSat as a member since 2017 and was chosen to be the team captain in 2019.

    During my time as captain, I led 30+ engineering students from multiple engineering disciplines to design and build the CanSat, as well as manage all the administrative work to maintain the team within the Ryerson community. I also managed all electronic subteams including Flight Software, Ground Software, PCB (printed circuit board) Design and Sensor Integration as well as Power. These subteams use MATLAB, C/C++ and Eagle CAD to build the electronic structure of the CanSat. In addition, I have organized several training sessions to familiarize new members with the basics of Arduino and Eagle CAD.

    Check out these articles about the team- Faculty of Engineering and Ryersonian!

2019

Ping Pong Game

    Created a Ping Pong game using VHDL to run on the Spartan 3E FPGA board by implementing custom logic circuits on the real-time high performance signal generators on the FGPA, and interfacing it with I/O devices such as the monitor and controller. This was a two-player game and controlled by the switches on the FPGA board. If the ball was missed and went out of bounds, then the ball turned red.

    The project is linked here!

Cache Controller

    Created a Cache Controller using VHDL on the Spartan-3E FPGA Board. The objective of this project was to implement a block responsible for copying code or data from main memory to cache memory automatically. It implemented static RAM (SRAM) and synchronous dynamic RAM (SDRAM) components as well as other custom logic controllers and memory units. The project was tested using the ChipScope Pro Tool by verifying the waveforms.

    The project is linked here!

2018

Banking Application

    Created a Banking App in Java and Java FX to replicate online banking transactions. Functionalities include creating new users accounts. These users could log in/out of their accounts as well as deposit/withdraw money from their account. All the accounts were also controlled by a manager that were able to log in/ log out into user accounts. Features that the Manager could control were adding/deleting customers and adding/withdrawing money from the account. Each customer was also given a rating based on the amount of money in their account- Gold, Silver and Platinum.
